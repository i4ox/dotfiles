# My dotfiles for DevOps station

## Tools

- GoEnv: Golang Version Manager
- NVM: Node Version Manager
- PyEnv: Python Version Manager

## Programs

- Browser: Firefox
- Editor: Neovim
- Terminal Emulator: Alacritty

## Terminal

- Zsh: shell
- Zap: zsh plugin manager
- Starship: theme for shell
- eza: best alternative for *ls*
- zoxide: best alternative for *cd*
